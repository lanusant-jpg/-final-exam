## 💡 프로젝트 소개
**Cinema Now**는 사용자가 복잡한 검색 없이 **실시간 영화 트렌드(순위)**를 한눈에 파악하고, 지인들과 정보를 손쉽게 공유할 수 있도록 개발된 안드로이드 애플리케이션입니다.

기존 영화 정보 서비스의 불편함을 해소하기 위해 **직관적인 UI**와 **원터치 공유 기능**에 중점을 두었습니다.

---

## ✨ 주요 기능
* ** 실시간 랭킹 조회:** 버튼 하나로 '일별 박스오피스'와 '주간 박스오피스'를 전환하여 조회
* ** 순위 시각화:** 1~3위 금/은/동 컬러 강조 및 전일 대비 순위 변동(🔺상승, 🔻하락, 🆕신규) 표시
* ** 포털 검색 연동:** '상세' 버튼 클릭 시 네이버 통합 검색으로 즉시 연결하여 정보 제공
* ** 간편 공유:** 영화 제목, 순위, 관객 수 정보를 카카오톡이나 문자로 친구에게 바로 전송

---

## 🛠 사용 기술 (Tech Stack)
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose (Material Design 3)
* **Architecture:** MVVM 패턴 (Model - View - ViewModel)
* **Network:** Retrofit2, Gson, Coroutines
* **Open API:** 영화진흥위원회(KOBIS) 박스오피스 API
